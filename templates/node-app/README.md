# node-app

Minimal Node.js starter within `ai-env`.

Structure
- `src/index.js` — module + CLI entry
- `test/index.test.js` — basic test using Node's built-in `assert`

Run
- Print greeting:
  - `npm run start --prefix projects/node-app -- AI`
- Run tests:
  - `npm test --prefix projects/node-app`

Notes
- No external dependencies required. Works with Node 18+ (ES modules).
